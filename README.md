Đề tài tập trung xây dựng hệ thống phân loại mức độ tổn thương khớp gối từ ảnh X-quang.
Triển khai và so sánh hai nhóm phương pháp chính:
- Nhóm mô hình học máy và học sâu truyền thống chạy trên máy tính cổ điển với dữ liệu gốc và dữ liệu sau xử lý.
- Nhóm mô hình học máy lượng tử và học sâu lượng tử được thực hiện trên môi trường mô phỏng máy tính lượng tử với dữ liệu gốc và dữ liệu sau xử lý.
